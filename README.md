# AC-X — клиент античита для Insurgency 2014

Скриншоты по запросу сервера, автообновление.

**Скачать:** [Releases](https://github.com/puma133/insurgency-acx/releases) — выберите последнюю версию и загрузите `insurgency_acx.exe`.

Рядом с exe положите `config.ini` (см. настройки сервера / у администрации).

Исходный код не публикуется.
